# Content Checklist: Salguero Lab Website

Use this checklist to track the collection, validation, and loading of content assets for the Salguero Lab Website.

---

## 1. Core Profile Details (Faculty/PI)
- [ ] High-resolution headshot of Dr. Salguero (saved to `assets/images/tina_salguero.jpg`)
- [ ] Biography (under 300 words)
- [ ] Education and professional background list
- [ ] Full contact info: office address, lab room numbers, phone, email
- [ ] Academic profile links (Google Scholar, ResearchGate, ORCID, UGA Directory)

## 2. Research Projects & Funding (data/projects.json & data/grants.json)
- [ ] Title and 3-sentence summary for each active research project
- [ ] High-quality representation image for each project (saved to `assets/images/projects/`)
- [ ] List of student researchers associated with each project
- [ ] Active and past grant lists (Funding Agency, Grant ID, Project Title, Timeline)

## 3. Team Roster (data/students.json & data/alumni.json)
- [ ] Full names and roles of all current graduate students
- [ ] Full names and roles of undergraduate assistants
- [ ] Student headshots (cropped square, saved to `assets/images/team/`)
- [ ] Custom bullet point for each current student describing their project focus
- [ ] Alumni list: Graduation year, degree (Ph.D./M.S./B.S.), and current position or destination

## 4. Publications List (data/publications.json)
- [ ] Complete citation text for all peer-reviewed journals
- [ ] Official DOI links
- [ ] Digital copies or links to PDFs (optional)
- [ ] Key research tags for each publication (e.g., *Nanosheets*, *Synthesis*, *Catalysis*)
- [ ] Abstract texts for the detailed view modal

## 5. Media & News Gallery (data/gallery.json & data/news.json)
- [ ] 5-10 high-quality photos of students working in the lab, group outings, or poster sessions
- [ ] Clear captions and photo credits for the gallery
- [ ] Content write-up for recent news items (e.g., student awards, publications, graduation, new grants)

## 6. Accessibility & Metadata Checklist
- [ ] Alt text written for every single image asset
- [ ] SVG icons contain accessible titles/descriptions
- [ ] Custom meta descriptions written for every main page
- [ ] Page-specific SEO title tags defined
